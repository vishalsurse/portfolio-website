01 — About

Infrastructure, automated end to end.

I'm Vishal Surse, an AWS Engineer at Webasha Technologies, working across the full delivery pipeline — from provisioning infrastructure with Terraform to shipping code through Jenkins and GitHub Actions, to watching it run in production through Grafana and Prometheus dashboards.

My work centers on making deployments boring — in the best way. That means blue-green rollouts with automated rollback, configuration that's version-controlled and repeatable, and monitoring that catches problems before they become incidents.
